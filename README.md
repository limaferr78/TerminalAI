# TerminalAI - Assistente Pessoal Android com IA Local 

Desenvolvimento de um aplicativo Android que executa um modelo de linguagem (LLM) baseado no Qwen de forma nativa.

Implementação de inferência local (sem dependência de APIs de nuvem), garantindo privacidade e funcionamento offline.
Respostas a comandos de texto e integração com ferramentas de busca para consultas híbridas (online/offline).
Java/Kotlin (Android SDK), Integração com Modelos Qwen, Prompt Engineering.

O modelo Qwen (1.1GB) não foi enviado ao GitHub por limitações de tamanho. Para rodar o projeto, baixe o arquivo qwen2.5-1.5b-instruct-q4_k_m.gguf e coloque na pasta assets.
